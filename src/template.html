Encrypt Sensitive Documents

Please use this page to encrypt sensitive documents. Only the party controlling the private keys will be able to view the contents of the encrypted file(s).

Step 1.

Put your files into a folder and compress the folder

To create a ZIP file from a folder:

On Windows, right click the folder and select "Send to" â€º "Compressed (zipped folder)".

On Mac, right click the folder and select "Compress [Folder Name]".

Step 2.

Choose your compressed folder by clicking "Choose File" and selecting it.

Step 3.

Click "Download" to download the encrypted file.

Step 4.

Email the encrypted file to the destination. You can recognize it by looking for the name of the original file/directory you encrypted, except it will have a ".gpg" extension.
